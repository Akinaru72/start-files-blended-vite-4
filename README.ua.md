# React Blended 4

## 1. Todos

- todos.items — масив усіх завдань
- підтримується:
  - додавання todo
  - видалення todo
  - редагування todo

---

## 2. Фільтр

- `filter` — рядок для пошуку
- використовується для фільтрації списку todo

---

## 3. Редагування

- `todos.currentTodo` — todo, яку зараз редагують

Логіка роботи:

- при натисканні кнопки **Edit** — поточна todo записується в `currentTodo`
- якщо `currentTodo !== null` — відображається форма редагування
- якщо `currentTodo === null` — відображається форма додавання

---

## 4. Форма редагування

- інпут заповнюється текстом поточної todo
- кнопка **Save**:
  - оновлює todo в `items`
  - встановлює `currentTodo` в `null`
- кнопка **Cancel**:
  - встановлює `currentTodo` в `null`

---

## Головна ідея

- `items` — всі todo
- `currentTodo` — todo, яку редагують
- `filter` — фільтр для списку
